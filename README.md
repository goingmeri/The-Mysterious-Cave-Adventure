# The Mysterious Cave Adventure

A text-based adventure game built with C# and .NET 8.0 that combines storytelling with a number guessing challenge.

## About

Wake up at the entrance of a mysterious cave and make choices that determine your fate. Will you enter bravely, investigate first, or run away? Each decision leads to different outcomes, treasures, and challenges.

## Features

- Multiple story paths and endings
- Interactive number guessing mini-game
- Colorful console output with emojis
- Choice-based gameplay
- Random elements for replayability

## Requirements

- .NET 8.0 SDK or later
- Windows, macOS, or Linux

## How to Run

1. Clone this repository
2. Navigate to the project directory
3. Run the following command:

```bash
dotnet run
```

## Gameplay

Make choices by entering numbers (1, 2, or 3) when prompted. Your decisions will lead you through different paths in the cave, each with unique challenges and rewards.

### Possible Outcomes

- Find treasure and magical items
- Solve riddles to progress
- Dodge ancient traps
- Discover hidden chambers
- Or play it safe and escape unharmed

## Project Structure

- `Program.cs` - Main game logic and story implementation
- `The-Mysterious-Cave-Adventure.csproj` - Project configuration file
